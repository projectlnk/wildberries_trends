# wildberries_trends
Go-сервис, который в реальном времени читает поисковые запросы из Kafka, ведёт точный подсчёт частоты за последние 5 минут и возвращает топ‑N запросов через HTTP API. Включает динамический стоп‑лист, метрики Prometheus, unit‑тесты, бенчмарки производительности и полностью готов к локальному запуску (Docker Compose + KRaft Kafka).
